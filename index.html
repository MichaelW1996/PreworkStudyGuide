HTML

  • The head element contains info about the webpage
  • The body element contains the visible content shown to a user
  • The body can be divided into child elements
  • HTML elements can be styled with metadata such as class or id to allow them to interact with CSS and JS implemented features

CSS

  • An asterix can be used as a wildcard or universal selector to apply styling to all elements unless more specific styling is applied conflictingly
  • A margin is the area around an element
  • Padding is the area inside the element around our content
  • multiple elements can be styled by seperating them with a comma
  • The "link" element is needed to connect our HTML file to our CSS file

Git

  • Git status - Shows the current banch
  • Git add -A = Adds all the files in the directory to a selection
  • Git commit -m = Takes the selection and bundles them together into a commit with a name (add name after -m)
  • Git push = pushes/uploads from local repo to github
  • Git pull = Pulls/Downloads from remote to local

Javascript

  • For loops can be used to repeat operations on different items in an array
  • numbering always starts at 0
  • A variable is a nambed container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
